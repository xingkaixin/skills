# line-sketch

简笔画：克制留白，单线涂鸦，无脸小人，底部中文手写引言

## Core Characteristics

极简手绘线稿风格。Off-white 纸张底色，深灰单色线条勾勒一切，唯一点缀色为哑光伞黄。构图大量留白，视觉焦点单一，概念先行而非写实。无脸圆头小人作为人物统一符号。整体基调克制、清醒、略带反讽，情绪平静。底部附一句短小的中文手写引言。

## Lines

- 深灰单色细线（#3A3A3A），不模拟笔压变化
- 线条有轻微自然颤动（非精准几何）
- 无填充色块，全线稿表达
- 人物统一为无脸圆头小人，圆形头 + 简笔身体

## Texture

- 轻微纸张纹理底色（off-white #F5F3EE）
- 无复杂纸面纹路，不喧宾夺主
- 无铅笔/炭笔磨损效果

## Depth

- 纯平面，无立体感、无阴影、无透视
- 通过元素大小和位置暗示层次
- 大量负空间即是深度表达

## Element Vocabulary

- 无脸圆头小人（人物唯一表达方式）
- 概念性单线图标（灯泡、箭头、对话框、圆圈等）
- 伞黄点缀（#D4A843）用于一处关键强调
- 底部短小中文引言（小字，自然手写体风格）
- 简单注释符号：箭头、下划线、括号

## Typography Approach

- 所有文字优先使用中文
- 手写体风格，笔画自然不规整
- 引言文字小而克制，位于画面底部
- 标题若出现，保持简洁、灰色、不抢主体焦点
